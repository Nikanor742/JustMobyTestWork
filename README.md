# JustMobyTestWork

![Gameplay](./Docs/Gameplay.gif)

Тестовое задание на Unity: небольшой FPS-прототип с врагами, несколькими типами оружия, прокачкой, сохранением прогресса и локализацией.

## Архитектура

- Код разбит по зонам ответственности: `Player`, `Weapons`, `Enemies`, `Game`, `Upgrades`, `Localization`, `Save`, `Configs`.
- Зависимости собираются через `VContainer` в `GameLTS`.
- Инициализация проекта проходит через `GameInitSystem`, который поднимает основные игровые системы.
- Данные игры вынесены в `ScriptableObject`: отдельно настроены игрок, оружие, враги, апгрейды, локализация.
- Логика отделена от представления: `Systems` отвечают за поведение, `Views` за Unity-объекты, `Models` за состояние, `Factories` за создание сущностей.

## Input System и тач-управление

Так как проект уже использует `Unity Input System`, на его основе можно достаточно просто добавить управление для тачскрина.

- завести отдельный action map для мобильного управления
- привязать виртуальный стик для движения и отдельную тач-зону для обзора камеры
- вынести стрельбу, прыжок и смену оружия на UI-кнопки, связанные с теми же input actions


## Конфиги

Все основные конфиги находятся в `Assets/Source/Configs`.

- `PlayerConfigSO` хранит параметры передвижения и обзора игрока
- `WeaponConfigSO` описывает тип оружия, урон, скорострельность, пулю и параметры выстрела
- `EnemyConfigSO` содержит настройки врагов: здоровье, скорость, лимит на сцене и тайминги
- `UpgradesConfigSO` задает тип улучшения и значения прогрессии по уровням
- `GameConfigSO` хранит общие параметры игры, например чувствительность мыши
- `LocalizationConfigSO` содержит языки и набор локализованных строк

## Локализация

Сейчас локализация хранится в `LocalizationConfigSO`, но в будущем ее можно расширить до пайплайна с Google Таблицей, чтобы удобнее редактировать переводы и синхронизировать текст без ручного заполнения данных в проекте.

Также при дальнейшем развитии локализации можно добавить отдельные шрифты под разные языки, чтобы корректно поддерживать наборы символов и избежать проблем с отображением текста.
